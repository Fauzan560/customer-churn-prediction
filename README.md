# 🛡️ Customer Churn Intelligence Suite
An end-to-end Machine Learning pipeline to predict customer attrition using XGBoost and Streamlit.

## 🚀 Overview
This project identifies high-risk customers in a telecommunications dataset. It moves beyond simple "black-box" predictions by using **SHAP** values to explain why a customer is likely to leave.

## 📊 Key Features
- **Predictive Engine:** Powered by XGBoost for high-accuracy tabular classification.
- **Explainable AI (XAI):** Integrated SHAP plots to visualize feature importance (Total Charges, Tenure, etc.).
- **Live Dashboard:** Interactive Streamlit UI for real-time risk assessment.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, Scikit-learn, XGBoost, SHAP, Streamlit
- **Deployment:** Streamlit Cloud / Docker
